Getting started with the SDK

Package overview

The IBM Navigator Java SDK package includes the following components:

The binary distribution.

  • See Downloading the SDK

  • The binary distribution contains the SDK jar itself and all of its dependencies that we have explicitly tested with this version.

  • Official Javadocs for the SDK

The Android sample

  • See Android Sample

  • The Android sample contains a working Android application built with the Navigator SDK. The application demonstrates how to do basic operations like authentication, browsing folders and documents as well as how to handle threading issues specific to Android.

The command line sample

  • The command line sample contains a command line Java program that you can compile and run inside of 5 minutes. The program uses connection information for a local ICN server that you provide in a properties file and is run with a single maven command (or Java command)

Watson integration command line sample

The Watson integration sample here shows how to apply the SDK as a source of data to the Watson Alchemy APIs.

Important: The sample projects are intended to show you how to use the SDK. It is not intended to be a tutorial on how to design your application. For example, it does not contain robust error handling and recovery.

SDK details

The SDK uses the Factory pattern and is organized into two sections: Factory: Factory methods to obtain all the actual model objects and perform high-level tasks. Model: The entire model layer that shows the protocols and interfaces that can be called.

For more details on the layout of the SDK and all of its classes see the Javadocs section link in the References section

Prerequisites

You are expected to have a working knowledge of IBM Content Navigator, and the Java programming language.

The prerequisites for using the SDK are as follows:

  • An IBM Content Navigator (2.0.3.5 or later) configured to use an IBM FileNet Content Manager server and/or IBM Content Manager server.
  • (Recommended) A recent version of Eclipse or Android Studio. Android studio is only required for building Android mobile applications.
  • The IBM Navigator Java SDK 1.0.0 or later package. See the download page for details on downloading the SDK.
  • Java Runtime Environment (JRE) 1.7 or later

Integration with your projects

To use the IBM Navigator Java SDK, you must integrate it with an existing or new project and this procedure will be different depending on whether the project is a desktop Java application or a Android mobile application.
