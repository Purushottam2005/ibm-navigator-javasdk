Overview

You can build high-quality desktop (non-mobile) and mobile apps for the Android platforms with the IBM Navigator Java Software Development Kit. This release supports both IBM FileNet Content Manager and IBM Content Manager repositories.

Here are the main goals of the software development kit (SDK):

  • High performance
  • Low memory requirements
  • Asynchronous operation
  • Encapsulation of implementation details like networking, serialization and security token management.
  • Expose the same object model as the iOS version of the SDK so that developers will feel comfortable moving between Java and Swift when needed.
  • Allow for the same binary Jar file to be used for both Desktop and Mobile development.

Here are some of the tasks that you can perform with the SDK:

  • Connect and log in to an on-premises IBM Content Navigator server.
  • View all documents and properties in a Navigator repository.
  • Add and remove documents in a Navigator repository.
  • Search for content in a Navigator repository.
  • Use search templates to search for content in a Content Navigator repository.
  • Use teamspaces to manage content and users.
  • Use favorites to quickly access content.
  • Sync content to keep it up to date.
